Teste do Widget de Chat

Esta página serve para testar o widget de Live Chat. O widget deve aparecer no canto inferior direito.

Configuração: Substitua SEU_TOKEN e SUA_API_URL abaixo pelo token público da caixa Live Chat e pela URL das funções Supabase (ex: https://SEU_PROJETO.supabase.co/functions/v1).

Se o widget não aparecer, verifique o console do navegador (F12) para mensagens de erro.
